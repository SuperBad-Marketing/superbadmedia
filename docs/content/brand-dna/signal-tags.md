# Brand DNA — Signal Tag Taxonomy

**Locked:** CMS-1 content mini-session, 2026-04-17.
**Spec ref:** `docs/specs/brand-dna-assessment.md` §5.
**Total:** 79 tags across 5 domains.

Each answer option maps to 1–3 tags. Frequency across all answers does the weighting — no manual weights. Higher frequency = stronger signal.

Supplement questions (Founder + Supplement track) map to `brand_override.<domain>.<tag>` — same tags, prefixed, so downstream prompts know "the founder is X but the brand is Y."

## Tag distribution notes

- Questions in any section can award tags from any domain. Personality traits especially surface across all five sections.
- The `brand_override` pseudo-domain for supplement questions uses the format `brand_override.<domain>.<tag>` (e.g. `brand_override.communication.formality`).
- Not every tag will appear in every completed profile. A tag at frequency 0 is itself informative — it's the absence of a signal.
- Natural tension pairs (e.g. `minimalism`/`maximalism`, `risk_appetite`/`risk_caution`, `introversion`/`extraversion`) create the interesting profiles. Contradictions are features, not bugs.
