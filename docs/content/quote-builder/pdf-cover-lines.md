# Quote Builder — PDF Cover Line Rotation Pool

> Canonical source for the dry Playfair italic cover line on Quote PDFs.
> Position 8 in the PDF template (spec §4.4). One line picked at random
> per PDF generation. Sprinkle claim: §3 in `docs/candidates/sprinkle-bank.md`.
>
> Decision: **rotation pool** (not Claude-per-quote). Locked CMS-2.
>
> Voice: dry, observational, restrained. These sit on a financial
> document — they're texture, not comedy. One line. One read. Move on.

## Usage rules

- One line per PDF. Random selection, no weighting.
- Lines are not client-specific — they work for any quote.
- Rendered in Playfair Display italic, Retro Pink (#F4A0B0), smaller
  than body text. Sits between the total block and the terms link.
- Pool can be extended post-launch. Minimum 20 at launch.
- No drift check needed (pre-approved pool, not generated).
