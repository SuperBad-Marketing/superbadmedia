# Setup Wizards — Completion Outro Lines

> Canonical source for the dry outro lines on the celebration step.
> One line picked per wizard completion. Stratified by wizard type and
> audience.
>
> Voice: client tone = branded, warm, company-for-the-task. Admin tone =
> terse, function-first.

---

## Selection rules

- One line per completion, picked from the matching category.
- If no category-specific line fits, fall through to General / fallback.
- Client tone: full sentence(s), warmth, texture.
- Admin tone: fragments. Andy sees these constantly — brevity is respect.
- No drift check needed (pre-approved pool).
